# Mini Code Editor

A lightweight, frontend-only code editor similar to CodePen, built with vanilla HTML, CSS, and JavaScript.

## Features

- **Three Code Editors**: Separate editors for HTML, CSS, and JavaScript
- **Live Preview**: Real-time preview in an iframe
- **Save & Load**: Store your code locally with LocalStorage
- **Auto-run**: Toggle automatic code execution
- **Keyboard Shortcuts**: Quick access to common actions
- **Responsive Design**: Works on desktop and mobile
- **Dark Theme**: Developer-friendly dark mode interface
- **Line Numbers**: Visual line numbering in editors

## How to Use

1. **Open** `index.html` in your web browser
2. **Write code** in the three editors:
   - HTML editor (left, orange icon)
   - CSS editor (middle, blue icon)
   - JavaScript editor (right, yellow icon)
3. **Click "Run Code"** to see the live preview
4. Use the control buttons:
   - **Run Code**: Execute and preview your code
   - **Clear All**: Empty all editors
   - **Save**: Store code locally
   - **Load Saved**: Restore saved code
   - **Auto-run**: Toggle automatic execution on code changes

## Keyboard Shortcuts

- `Ctrl + S` - Save code
- `Ctrl + L` - Load saved code
- `Ctrl + R` - Run code
- `Tab` - Insert 2 spaces (in editors)
